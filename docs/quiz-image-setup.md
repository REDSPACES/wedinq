# クイズ画像セットアップガイド

## 画像の配置場所

クイズの問題画像は以下のディレクトリに配置してください：

```
/public/quiz-slides/
```

## ファイル名の規則

ファイル名は以下の形式で命名してください：

```
question-1.jpg  ← 第1問
question-2.jpg  ← 第2問
question-3.jpg  ← 第3問
question-4.jpg  ← 第4問
question-5.jpg  ← 第5問
```

## 画像の仕様

### 必須要件
- **アスペクト比**: 16:9（例: 1920×1080px）
- **ファイル形式**: JPG または PNG
- **ファイル名**: `question-{番号}.jpg`（1〜5）

### 推奨仕様
- **解像度**: 1920×1080px 以上
- **ファイルサイズ**: 500KB〜2MB
- **色空間**: sRGB
- **品質**: 高品質（圧縮率70〜85%）

## 画像の内容構成

各問題画像には以下を含めてください：

1. **問題文**: 大きく読みやすいフォント
2. **選択肢**: A、B、C、Dの4つ（必須）
3. **装飾**: ウェディングテーマに合ったデザイン

### レイアウト例

```
┌──────────────────────────────────────────┐
│                                          │
│  第1問                                   │
│                                          │
│  新郎の趣味は次のうちどれでしょう？      │
│                                          │
│  A. 釣り                                 │
│  B. ゴルフ                               │
│  C. 読書                                 │
│  D. サーフィン                           │
│                                          │
└──────────────────────────────────────────┘
```

## Canvaでの作成手順

### 1. 新規プロジェクト作成
- Canvaにログイン
- 「プレゼンテーション（16:9）」を選択
- サイズを確認: 1920×1080px

### 2. デザイン作成
- 背景色/背景画像を設定（ウェディングテーマ）
- 問題文を追加（見出しスタイル、大きめのフォント）
- 選択肢A〜Dを追加（読みやすいフォント）
- 必要に応じて装飾を追加

### 3. 5問分のスライドを作成
- ページを追加して各問題を作成
- 一貫性のあるデザインを維持

### 4. ダウンロード
- 「ダウンロード」→「JPG」を選択
- すべてのページをダウンロード
- ファイル名を `question-1.jpg`〜`question-5.jpg` に変更

### 5. 配置
- ダウンロードしたファイルを `/public/quiz-slides/` に配置

## デザインのポイント

### 読みやすさ
- フォントサイズ: 問題文は48pt以上、選択肢は36pt以上推奨
- コントラスト: 背景と文字のコントラストを高く保つ
- 余白: 十分な余白を確保し、情報を詰め込みすぎない

### 色使い
- **高コントラスト**: 会場のスクリーンでも見やすい色合い
- **統一感**: 5問すべてで配色を統一
- **ブランドカラー**: 新郎新婦のテーマカラーを使用

### レイアウト
- **中央寄せ**: 重要な情報は画面中央に配置
- **安全マージン**: 画面端から10%程度の余白を確保
- **順序**: 問題文 → 選択肢の順で視線が流れるように配置

## テスト方法

### 1. 画像を配置後、ブラウザで確認
```
http://localhost:3001/screen
```

### 2. 動作確認
- 「クイズを開始」をクリック
- 第1問の画像が表示されることを確認
- 「結果を表示」→「次の問題へ」で第2問に進むことを確認
- すべての問題が正しく表示されるか確認

### 3. 画像が表示されない場合
- ファイル名が正しいか確認（`question-1.jpg`など）
- ファイルが `/public/quiz-slides/` に配置されているか確認
- ブラウザをリロード（Cmd/Ctrl + R）

## トラブルシューティング

### 画像が表示されない
- **原因**: ファイル名が間違っている
- **解決**: ファイル名を `question-{番号}.jpg` に変更

### 画像が歪んで表示される
- **原因**: アスペクト比が16:9でない
- **解決**: 画像を1920×1080pxにリサイズ

### 画像の読み込みが遅い
- **原因**: ファイルサイズが大きすぎる
- **解決**: 画像を圧縮（2MB以下を推奨）

## サンプル画像

サンプル画像を作成する場合は、以下のテンプレートを使用してください：

### テンプレート内容
- 背景: ピンク〜ホワイトのグラデーション
- 問題文フォント: Playfair Display または 游明朝
- 選択肢フォント: Noto Sans JP または ヒラギノ角ゴ
- 問題番号: 左上に配置
- 選択肢: 縦に配置（A, B, C, D）

## 正解データの設定

画像とは別に、正解データを設定する必要があります。
詳細は別途、Supabase設定ガイドを参照してください。

正解データの例：
```typescript
const questions = [
  { id: 1, correctAnswer: 0 }, // A が正解
  { id: 2, correctAnswer: 2 }, // C が正解
  { id: 3, correctAnswer: 1 }, // B が正解
  { id: 4, correctAnswer: 3 }, // D が正解
  { id: 5, correctAnswer: 0 }, // A が正解
];
```
