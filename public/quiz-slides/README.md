# クイズスライド画像配置ガイド

## 画像の配置方法

このディレクトリに、クイズの問題画像を配置してください。

### ファイル名規則

```
question-1.jpg  ← 第1問の画像
question-2.jpg  ← 第2問の画像
question-3.jpg  ← 第3問の画像
question-4.jpg  ← 第4問の画像
question-5.jpg  ← 第5問の画像
```

### 画像仕様

- **アスペクト比**: 16:9（必須）
- **推奨解像度**: 1920×1080px 以上
- **ファイル形式**: JPG または PNG
- **ファイルサイズ**: 500KB〜2MB程度を推奨
- **内容**: 問題文 + 選択肢（A, B, C, D）

### 画像作成のポイント

1. **問題文**: 大きく読みやすいフォントで
2. **選択肢**: A、B、C、Dの4択を明確に表示
3. **色使い**: 会場のスクリーンで見やすい色合い（高コントラスト推奨）
4. **レイアウト**: 情報が中央寄せで、端が切れないように

### 例：画像の構成

```
┌─────────────────────────────────────┐
│                                     │
│    第1問                            │
│                                     │
│    新郎の出身地はどこでしょう？      │
│                                     │
│    A. 東京都                        │
│    B. 大阪府                        │
│    C. 福岡県                        │
│    D. 北海道                        │
│                                     │
└─────────────────────────────────────┘
```

### Canvaでの作成例

1. Canvaで「プレゼンテーション（16:9）」テンプレートを選択
2. サイズ: 1920×1080px
3. 各問題を1スライドとして作成
4. JPG形式でダウンロード
5. ファイル名を `question-1.jpg` などに変更
6. このディレクトリに配置

## 画像が配置されるまで

画像が配置されるまで、画面にはプレースホルダーが表示されます。
